## Name of The Project ##
Quizzards
## AUTHOR ##

## DESCRIPTION ##
The project lets the user choose answers from a group of multiple choice questions and returns the appropriate grade.
## BDD ##
|  Input               |Output|
|---     |---|
| two correct answers  | 50  |
| three correct answers| 75  |
| four correct answers | 100 |
